CentOS7安装Nginx

2020-05-26 Software_Install CentOS7, Nginx 评论

1. 下载Linux版的Ngnix

下载链接

2. 上传文件到Linux服务器

使用lrzsz方式上传,直接拖拽

3. 解压上传文件,并删除压缩包

# 解压文件
[root@localhost src]# tar -xzvf nginx-1.19.0.tar.gz 
nginx-1.19.0/
nginx-1.19.0/auto/
......
nginx-1.19.0/auto/cc/name
nginx-1.19.0/auto/cc/owc
nginx-1.19.0/auto/cc/sunc
# 查看文件目录结构
[root@localhost src]# ls
images  jdk1.8  nginx-1.19.0  nginx-1.19.0.tar.gz  tomcats
# 删除压缩包
[root@localhost src]# rm -rf nginx-1.19.0.tar.gz

  1. 执行安装指令
[root@localhost nginx-1.19.0]# make install

...
make[1]: 离开目录“/usr/local/src/nginx-1.19.0”
  1. 检查安装路径
# 工作目录
[root@localhost nginx-1.19.0]# whereis nginx
nginx: /usr/local/nginx
  1. 安装中报错信息:
#安装Nginx时报错

./configure: error: the HTTP rewrite module requires the PCRE library.

#安装pcre-devel解决问题
yum -y install pcre-devel

#错误提示：
./configure: error: the HTTP cache module requires md5 functions
from OpenSSL library.   You can either disable the module by using
--without-http-cache option, or install the OpenSSL library into the system,
or build the OpenSSL library statically from the source with nginx by using
--with-http_ssl_module --with-openssl= options.

# 解决办法：
yum -y install openssl openssl-devel

# 总结：
yum -y install pcre-devel openssl openssl-devel

./configure --prefix=/usr/local/nginx

make

make install

5. Nginx命令

  1. 进入工作目录,查看目录结构
