Docker 安装 MongoDB

2020-05-24 Software_Install Ubantu,MongoDB 评论

4. 创建存储目录

创建MongoDB持久化文件目录。

mkdir -p /home/data/mongo/data

5. 启动容器

options说明:

  • –restart=always: 重启策略
  • -d: 后台运行容器，并返回容器ID
  • -p: 端口映射，格式为：主机(宿主)端口:容器端口
  • –name: 为容器指定一个名称
  • -v: 给容器挂载存储卷，挂载到容器的某个目录
root@Aspire:~# docker run --restart=always -p 27017:27017 --name mongo -v /home/data/mongo/data:/data/db -d mongo --auth
#启动后的信息
